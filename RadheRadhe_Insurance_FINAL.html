राधा राधा Insurance — Online Setup

Sirf ek baar karo — 10 minute mein ready | Bilkul Free

📋 Kya karna hai: Firebase (Google ka free cloud) mein account banao → Config copy karo → Yahan paste karo → Done! Sab users real-time data share karenge.
1
Firebase Project Banao (Free)
1. 🔗 console.firebase.google.com kholein (Google account se login karein)
2. "Add project" click karein → Name: radhe-radhe-insurance → Continue
3. Google Analytics → Disable → "Create project"
4. Project ready hone ke baad "Continue"
2
Authentication Enable Karo
1. Left sidebar mein "Build" → "Authentication"
2. "Get started" → "Email/Password" → Enable → Save
3. "Users" tab → "Add user":
  • Admin: [email protected] → apna password
  • User1: [email protected] → apna password
  • (Aur users baad mein bhi add kar sakte ho)
3
Firestore Database Banao
1. Left sidebar → "Firestore Database" → "Create database"
2. "Start in test mode" select karo → Next → Location: asia-south1 → Enable
3. Database ready ho jayega (1-2 minute)
4
User Roles Setup (Admin/User)
1. Firestore mein "Start collection" → Collection ID: users → Next
2. Admin ke liye: Document banao:
  email: [email protected]
  name: Administrator
  role: admin
3. User ke liye: + Add document:
  email: [email protected]
  name: Data Entry User
  role: user
5
Firebase Config Copy Karo — Yahan Paste Karo
1. Firebase Console → ⚙️ Project Settings (top-left gear icon)
2. "Your apps" section → "Add app" → Web icon </> → App nickname: CRM → Register
3. firebaseConfig object dikhega — neeche paste karein:
apiKey: "AIzaSy..."
authDomain: "your-project.firebaseapp.com"
projectId: "your-project-id"
storageBucket: "your-project.appspot.com"
messagingSenderId: "123456789"
appId: "1:123:web:abc123"
6
Website Host Karo (Free — Netlify)
Config save karne ke baad yeh file download hogi → phir:
1. 🔗 app.netlify.com/drop kholein
2. HTML file drag-drop karein → Deploy!
3. Aapko ek free URL milega: https://random-name.netlify.app
4. Woh URL 2-3 users ko share karein → sab ek saath use kar sakte hain!
⚠️ Important: Config save karne ke baad jo file download hogi, wahi Netlify mein upload karna. Yeh current file nahi — woh neeche button se banegi.
